# Medusa GPT 🪼

Asistente inteligente con interfaz de medusa animada.

## Estructura del Proyecto
- `server/`: Backend en Node.js (Express).
- `index.html`: Punto de entrada frontend.
- `script.js`: Lógica de animación y chat.
- `style.css`: Estilos visuales.

## Configuración
1. Instalar dependencias: `npm install`
2. Crear archivo `.env` con su `OPENAI_API_KEY`.
3. Iniciar servidor: `npm start`
